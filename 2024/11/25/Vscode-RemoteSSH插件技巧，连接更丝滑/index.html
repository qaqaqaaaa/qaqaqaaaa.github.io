使用vscode时，难免会遇到远端开发的情形，这时候需要通过remote ssh远程连接服务器进行开发，这里介绍几个技巧让远程连接更丝滑

插件本地下载

开启此项配置，直接跳过远程下载的环境，使用本地网络进行下载，以提高下载速度

本地下载VS Code Server

remote ssh连接服务器时需要在服务器伤下载一个大小为几十m的包，可以在此处查看log

正常无配置下的流程是首先使用远端服务器进行此包的下载，下载失败则使用本地网络下载，然后scp到远端，这里如果远端服务器网络无外网条件可以如下配置，可以提高速度
