介绍

ZeroTier 是一个高性能的开源虚拟网络平台，旨在提供简单、高效、安全的虚拟局域网（LAN）和虚拟广域网（WAN）解决方案。它结合了软件定义网络（SDN）和虚拟专用网络（VPN）的功能，允许用户跨越地理区域，将多个设备连接到一个统一的虚拟网络中，就像它们在同一个本地网络中一样。

核心功能

  1. 虚拟网络创建
    • ZeroTier 支持用户创建私有虚拟网络（类似于传统 VPN），将分散的设备连接起来，无需复杂的硬件配置。
    • 虚拟网络可以模拟二层或三层网络（如以太网或 IP 网络）。
  2. 跨平台支持
    • 支持多种操作系统，包括 Windows、macOS、Linux、iOS、Android 等，甚至可以嵌入到嵌入式系统中（如树莓派）。
  3. P2P（点对点）连接
    • 使用点对点（P2P）通信技术，ZeroTier 优先通过直接连接传输数据，从而降低延迟、提高速度。
    • 在无法直接连接时，会使用 NAT 打洞或中继服务器来保持连接。
  4. 易用性
    • 配置简单，只需几分钟即可完成安装和部署。
    • 提供用户友好的图形界面和命令行工具。
  5. 安全性
    • 使用端到端加密（AES-256）和认证技术，确保数据传输的安全。
    • 支持访问控制和权限管理，通过 ZeroTier Central 管理网络成员。
  6. 灵活性
    • 支持 IP 分配、路由、自定义 DNS 和其他高级网络配置。
    • 可用于跨云、混合云和物联网（IoT）环境。

ZeroTier 与传统 VPN 的对比

特性ZeroTier传统 VPN
配置复杂度简单（无需额外硬件）通常较复杂（需要配置服务器）
连接模式点对点（优先直连）中继传输
性能高效，低延迟延迟较高
安全性端到端加密依赖服务器配置
灵活性支持多种应用场景功能单一，针对远程访问

原理

zerotier可以通过两种方式进行连接，一种是中继传输，一种的p2p直连（通过udp方式进行打洞），优先都是采用p2p直连的方式，在打洞失败的情况下则会使用中继传输。

而区分局域网的方式则是通过一个网络的概念，在zerotier官网可以进行登录然后创建网络，然后不同的设备都加入这个网络。中继服务器在zerotier中是作为PLANET出现，这是zerotier自带的根服务器，也是打洞所需要，但是由于根服务器在国外会遇到网络问题，因此这时候可以在国内搭建中继服务器进行加速，这在zerotier中的概念为MOON，而普通的用户设备则被称之为LEAF

简单的说，MOON连接PLANET LEAF会连接MOON或者PLANET，MOON和PLANET称之为中继服务器提供打洞或者中继的服务。

官网的解释

we introduced the ability to add your own user-defined roots. Since the data center we inhabit is the planet, a user-defined set of roots is called a moon. When a node “orbits” a moon, it adds the moon’s roots to its root server set. Nodes orbiting moons will still use planetary roots, but they’ll use the moon’s roots if they look faster or if nothing else is available.
